用下列方法，我清除了50G左右的内存
![[屏幕截图 2026-01-20 093637.png]]
删除原因：prefetch，temp，Windows缓存文件等都是系统运行时产生的临时，缓存类文件，删除不影响电脑系统正常运行，以下是具体实操方法。

1.删除```prefetch``` 文件夹所有文件,```win``` + ``` R```,  输入 ```prefetch``` ,点确定或```Enter``` ，进入prefetch文件夹， ```ctrl``` +```A``` 全选所有文件，右键点击删除

```bash
prefetch
```

2.删除```temp ``` 文件夹所有文件，```win``` + ``` R```,  输入```temp ``` ，点确定或```Enter``` ,进入temp文件夹， ```ctrl``` +```A``` 全选所有文件，右键点击删除

```bash
temp
```

3..删除```%temp% ``` 文件夹所有文件，```win``` + ``` R```,  输入```%temp%``` ,点确定或```Enter``` ,进入%temp%文件夹， ```ctrl``` +```A``` 全选所有文件，右键点击删除

```bash
%temp%
```

桌面底部打开任务栏设置，选择 ```系统``` - ```存储``` - ```临时文件```，勾选文件均可删除，选择后点删除文件
